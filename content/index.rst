AIS Graph Classification
========================

Graph Neural Networks for Maritime Vessel Activity Classification

This tutorial demonstrates how to use Graph Neural Networks (GNNs) to classify
vessel fishing activities from Automatic Identification System (AIS) time-series
data, using the Deep Graph Library (DGL).

**Key Result:** GraphSAGE achieved **94.4% test accuracy** on the fishing vs.
non-fishing classification task.
